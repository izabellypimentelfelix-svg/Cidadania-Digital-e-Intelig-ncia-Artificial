# Portal Consciência IA 🌐🤖

## 📋 Escopo e Objetivo do Projeto
Este portal é uma aplicação Web estática projetada com o objetivo de servir como uma central de conscientização comunitária. Alinhado com as discussões contemporâneas sobre segurança cibernética, o site capacita os cidadãos a reconhecer e mitigar os efeitos das **Deepfakes** e de ecossistemas automatizados de **Desinformação**, promovendo uma postura ética e cidadã no espaço digital.

Desenvolvido para atender aos critérios de recuperação da disciplina de **Educação Digital e IA (Ensino Médio)**.

---

## 🚀 Arquitetura Técnica e Recursos Implementados
* **Acessibilidade Nativa (Subcategoria B):** Alternância em tempo real entre interfaces clara e escura utilizando herança de escopo em variáveis `:root` do CSS.
* **Design Responsivo Avançado:** Utilização de arquitetura Flexbox e regras baseadas em Media Queries para adaptação perfeita em dispositivos móveis.
* **Validação Interativa (Subcategoria A):** Script dinâmico em JavaScript puro (Vanilla JS) que extrai dados capturados em inputs de formulário, processa matrizes de acertos e falhas e injeta feedbacks customizados no DOM.

## 🎯 Objetivo do Portal
Este site foi desenvolvido como um portal de conscientização comunitária focado em um dos debates mais urgentes da atualidade tecnológica e social: **O Impacto das Deepfakes e da Desinformação na Sociedade**. 

O objetivo principal é educar os utilizadores a identificar mídias manipuladas por IA, alertar sobre os perigos da desinformação automatizada e fornecer interatividade prática através de artigos informativos, uma galeria de análise visual e um mini-game dinâmico.

---

## 🛠️ Tecnologias e Arquitetura do Projeto

O projeto foi construído do zero utilizando tecnologias web nativas, separadas rigidamente para garantir a organização e performance:

* **`index.html` (Estrutura):** Construído inteiramente com tags semânticas (`header`, `main`, `section`, `article`, `footer`) para garantir acessibilidade, além de elementos interativos de formulário (`form`, `input`, `button`).
* **`style.css` (Design e Layout):** Responsável pela identidade visual. Utiliza **Flexbox** e **CSS Grid** para alinhar os artigos e a galeria. Contém variáveis nativas (`:root`) para suportar a troca de temas e **Media Queries** para total responsividade em dispositivos móveis.
* **`script.js` (Interatividade):** Controla a lógica do sistema. Altera dinamicamente o DOM para o botão de acessibilidade (Modo Escuro) e processa as variáveis e validações do jogo "Detetive da Verdade", atualizando o placar em tempo real.

## 🤖 Prompts de IA Utilizados (Documentação)

Como parte do processo de desenvolvimento e refinamento de código, foram utilizados os seguintes comandos estruturados com inteligência artificial:

1.  *Prompt para Arquitetura HTML:* > "Crie a estrutura semântica de um site sobre cidadania digital e deepfakes, contendo seções para artigos, galeria de imagens e um formulário para um quiz/jogo interativo usando tags HTML5 puras."
2.  *Prompt para Estilização Avançada:* > "Desenvolva um arquivo CSS moderno que organize uma seção de artigos em grid (3 colunas em telas grandes, 1 coluna no mobile) e use Flexbox para alinhar uma galeria de fotos e um formulário de jogo. Adicione suporte a um modo escuro via classe .dark-mode."
3.  *Prompt para Lógica JavaScript:* > "Escreva um script JS nativo que capture o evento de clique de um botão de alternância de tema (claro/escuro) e valide as respostas de um input de rádio de um jogo, somando 10 pontos ao placar caso a resposta esteja correta e exibindo um feedback estilizado na tela sem usar alert."
